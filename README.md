Project Description
This project is a static e-commerce landing page for "EVOLVE CRAZY OUTFIT MENS WEAR," showcasing a modern and trendy men's clothing collection. The design is sleek, utilizing a dark, high-contrast theme (dark background with bright blue accents) to give it a futuristic and premium feel.

The page is built entirely with HTML5 and CSS3, focusing on clean structure, responsiveness (using minmax in the product grid), and clear user interaction points.

Key Features
Modern Dark Theme: A visually engaging color scheme (#0d0d0d background, white text, and #0af / #008cff blue accents) for a contemporary look.

Responsive Product Grid: Uses CSS Grid (display: grid and repeat(auto-fit, minmax(230px, 1fr))) to ensure the product cards adapt gracefully to different screen sizes.

Interactive Navigation Bar: Features prominent buttons for Location, Contact, Instagram, and Whatsapp for quick access and customer engagement.

Hover Effects: Implements smooth transitions and scaling (transform: scale(1.05)) on navigation buttons and product cards to enhance user experience.

Clear Call-to-Actions (CTAs): Dedicated "Buy Now" buttons for each product section.
